# 01.epics — FabTasks Capability Streams (v1.0)

Epics vormen de hoogste strategische laag binnen het FabTasks-project. Ze zijn FabOps capability streams die structureel volwassen moeten worden.

## 1. Doel van epics
- Structurele capability
- Multi-sprint strategische lijn
- Competentie voor hergebruik
- Bron voor stories en tasks

## 2. FabOps workflow
EPIC → STORY → TASK → BUILD/TEST/LOG/COMMIT

## 3. FabTasks Epics
- FT-EP-FOUNDATION
- FT-EP-DEVICE
- FT-EP-FIRMWARE
- FT-EP-BACKEND
- FT-EP-DESKTOP
- FT-EP-RELEASE

## 4. Structuur per epic
- Purpose
- Goals
- Scope
- Out of Scope
- KPIs
- Capability Level (C0–C5)
- Links

## 5. Lifecycle
Initiation → Growth → Maturity → Sustainment

## 6. Capability Levels
C0 → C5 zoals beschreven.

## 7. Updatefrequentie
Maandelijks + bij milestones.

## 8. Best Practices
- Epics voor richting, niet microplanning.

## 9. Gerelateerde bestanden
stories, tasks, sprints, manifest.

## 10. Versiegeschiedenis
v1.0 initieel.
